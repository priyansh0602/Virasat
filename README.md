# 🏛️ Virasat — RAG-Driven Multilingual Heritage Knowledge Base

A production-grade full-stack app built with **React + Vite**, **Tailwind CSS**, **Supabase**, and the **Wikipedia MediaWiki API**.

---

## ✨ Features

| Feature | Details |
|---|---|
| **Auth** | Email/Password signup, Google OAuth, Email OTP (6-digit) |
| **Onboarding** | Indian State dropdown + City input saved to Supabase user metadata |
| **Heritage Search** | Global Wikipedia full-text search (`action=query&list=search`) |
| **Geosearch** | Local heritage sites via Wikipedia Geosearch API within 15km of your city |
| **Storytelling Cards** | Cards with title, extract, thumbnail, "Read More" & "Generate Story" |
| **Language Toggle** | English / हिन्दी / संस्कृतम् |
| **RAG Placeholder** | "Storytelling Engine" section — ready to wire up to Claude/Gemini

---


## 🛠️ Tech Stack

- **React 18** + **Vite 5**
- **Tailwind CSS 3** (custom heritage color palette)
- **Supabase JS v2** (Auth: email, Google OAuth, OTP)
- **Wikipedia MediaWiki REST API** (search + geosearch + extracts + images)
- **Lucide React** icons
- **Google Fonts**: Playfair Display, Cormorant Garamond, Lato
